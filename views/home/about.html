the RESTful(ish) taylor swift api

What even..?

baelor.io - is a fully RESTful(ish) API for consuming data on the most important artist of our generation.

Taylor Swift.

Why even..?

Want to make taylor_swift_ebooks? Want to get the producers and writers of Shake It Off? Now you can do that with a simple HTTP request to baelor.io.

truly distuptive

What is this running on?

  • The API is built on golang, querying from a MySQL database. Inside a nice couple of Docker containers.

But.. The data?

The album, album cover, and song data was scraped from wikipedia. The lyric data was scraped from a lyric wiki. All data is owned by bae.
